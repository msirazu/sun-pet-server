# 🐾 Sun Pet - Server Side

This is the backend server for the Sun Pet application, a MERN-stack pet adoption platform. It handles API requests, database operations with MongoDB, and secure authentication.

## 🚀 Key Features

1. REST API: Built with Express.js to handle all frontend requests.

2. Database: MongoDB.

3. Authentication: Secure user login/registration with BetterAuth.

4. CRUD Operations: Managing pets, user adoption requests, and listing management.

5. Security: Environment variables for sensitive keys and CORS middleware configuration.

## 🛠 Tech Stack

1. Runtime: Node.js

2. Framework: Express.js

3. Database: MongoDB (Atlas)

4. Authentication: BetterAuth

## 📦 Main Dependencies

1. express

2. dotenv

3. cors
